# ccViewer

## how to compile
if you did not set up, run these commands.

```bash
brew install mercurial
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer/
brew install nasm
brew install yasm
```

### prepare depencency

```bash
cd work

./ffmpag_compile.sh
```
